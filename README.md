# AES Encryption and Decryption

This is a C implementation of the Advanced Encryption Standard (AES). I developed this project to understand how symmetric encryption works and to practice cryptographic concepts.

## Features

- Encryption of plain text using the AES algorithm
- Decryption of encrypted text back to its original form
- Plain and readable source code focused on core logic

## How to Run It

You can compile and run the project using gcc or any C IDE.

### Compilation

Open your terminal in the project directory and run:

```bash
gcc aes.c -o aes_app
